MeteoInfo 1.2.2 was released (2015-5-6)

MeteoInfo 1.2.2 was released. A new application of MeteoInfoLab is created with very first stage status. It was designed as a scientific computation and plot software mainly with console and script interface similar with MatLab and Spider. Development language are Java and Jython base on MeteoInfo library.
