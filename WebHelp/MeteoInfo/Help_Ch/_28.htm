显示站点数据

站点数据能够被显示成如下图形类型：

²        Station Point （点图）

²        Station Info （点图，所有字段添加入属性数据）

²        Contour （等值线，插值计算）

²        Shaded （等值线填色，插值计算）

²        Barb （风向杆）

²        Weather Symbol （天气现象）

²        Station Model （站点填图）
