TrajStat 1.4.4 was released (2015-2-11)

TrajStat 1.4.4 was released. The weighting function of PSCF/CWT according trajectory number was added.

The weighting method by endpoint number can not reduce the PSCF/CWT value of the grid cell which only has one or few trajectories with high observation value but relative big endpoint number due to the slow air mass speed.


So the trajectory number weighting function was added to reduce the PSCF/CWT value at the cells with few trajectories.

A 'N_Traj' field was added in the created PSCF/CWT layer to record the trajectory number in each grid cell, which can be obtained with 'Nij' at same time by clicking 'Get Nij' button in PSCF/CWT analysis dialog forms.
