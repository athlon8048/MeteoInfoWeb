MeteoInfo script tutorial 16: Grid data calculation

Data analysis is an important function in MeteoInfo implemented by grid and station data calculation. Unfortunately, Java does not support operator override, so the methods of "add", "sub", "mul" and "div" were added in GridData class to do arithmetic. The DataMath class in "org.meteoinfo.data" was designed to do more complex calculation, such as exponent, power, trigonometric functions. But the formula looks not so concise. For example, the formula like this:
es = 6.112*exp(17.67*(air-273.16)/(air-29.65))

should be written as:
es = DataMath.exp(air.sub(273.16).mul(17.67).div(air.sub(29.65))).mul(6.112)
